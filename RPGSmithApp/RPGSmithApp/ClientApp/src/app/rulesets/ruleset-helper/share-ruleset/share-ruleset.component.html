Share Rule Set

Input the recipient’s email address in the field below for whom you would like to share this Rule Set.

Shared Rule Sets are copied to the recipients account when they input a custom code generated and associated with the source rule set at the time they enable sharing. Any future changes to the source Rule Set will not affect other users.

All content is duplicated at the time of installation by the recipient, except for images. Any existing image references at the time the copy occurs will continue to use images stored on the source user’s account. Should the source user delete the Rule Set or remove images the recipients Rule Set will revert to a default stock image.
